Installation

1. Installation via NPM

This project has been created with Angular 6+, it is compatible with Angular 7. It is using for dependencies ng-bootstrap, font-awsome and moment.

the project is available on Github and installation using npm 

    npm install --save @biliboobrian/ng-datasheet @ng-bootstrap/ng-bootstrap @fortawesome/fontawesome-free moment moment-es6
