Welcome to Ng-DataSheet

All-in-one Angular 6+ inline spreadsheet

Ng-DataSheet is a free project proposing an extensible datagrid with or without inline editing.

Installation guide here

Enter or Double click : Switch to edition mode, switch to read mode on new line first selected column
Tab : Go to next column, if you reach end of selectable column in table, go to next row first column
Arrow : Navigate in cells
Shift + Arrow : Make a selection
Ctrl + C : Copy mouse selection (can be paste on spreadsheet applications)
Ctrl + V : Paste data in cell(s) (ca be copied from spreadsheet applications)
Esc : Remove current modifications
Home : Go to the first cell of the first row
End : Go to the last cell of the last row
