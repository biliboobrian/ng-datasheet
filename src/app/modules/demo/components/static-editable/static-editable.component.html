Exemple of static data with editable ng-datasheet

This Exemple show you a editable datasheet with a finish set of data with filter, sort and resizing capabilities. You can also add new row at the end of the table.
Enter or Double click : Switch to edition mode
Tab : Go to next column, if you reach end of table, go to next row first column
Arrow : Navigate in cells
Ctrl + C : Copy mouse selection (can be paste on spreadsheet applications)
Ctrl + V : Paste data in cell(s) (ca be copied from spreadsheet applications)
Esc : Remove current modifications
Home : Go to the first cell of the first row
End : Go to the last cell of the last row
