NgDatasheet

Component

The NgDatasheet is the main component to instanciate a grid.

Selector ds-datasheet

Exported as NgDatasheetComponent

Inputs

activeIds

An array or comma separated strings of panel identifiers that should be opened

Type: string | string[]
closeOthers

Whether the other panels should be closed when a panel is opened

Type: boolean
Default value: false — initialized from NgbAccordionConfig service
destroyOnHide

Whether the closed panels should be hidden without destroying them

Type: boolean
Default value: true
type

Accordion's types of panels to be applied globally. Bootstrap recognizes the following types: "primary", "secondary", "success", "danger", "warning", "info", "light" and "dark

Type: string
Default value: - — initialized from NgbAccordionConfig service

Outputs

panelChange

A panel change event fired right before the panel toggle happens. See NgbPanelChangeEvent for payload details

Methods

isExpanded

isExpanded(panelId: string) => boolean

Checks if a panel with a given id is expanded or not.
expand

expand(panelId: string) => void

Expands a panel with a given id. Has no effect if the panel is already expanded or disabled.
expandAll

expandAll() => void

Expands all panels if [closeOthers]="false". For the [closeOthers]="true" case will have no effect if there is an open panel, otherwise the first panel will be expanded.
collapse

collapse(panelId: string) => void

Collapses a panel with a given id. Has no effect if the panel is already collapsed or disabled.
collapseAll

collapseAll() => void

Collapses all open panels.
toggle

toggle(panelId: string) => void

Programmatically toggle a panel with a given id. Has no effect if the panel is disabled.
