Column

Class

Definition of a column in your datasheet.

Properties

title

Title printed has header on the column.

Type: String
data

Property of the dataset Object to show on this column.

Type: String
width

Defaut width of the column (overrided by user preset if dsKey is st to true on ng-datasheet component).

Type: Number
Default value: 0
noWidth

When another column is resized, column mark as noWidth is resized to do not adapt all columns.

Type: Boolean
Default value: false
options

Set of options used to init Cell view / edit renderer.

Type: Options
neededForAdd

Column needed to add new object in dataSet.

Type: Boolean
Default value: true
searchable

Activate filter capability.

Type: Boolean
Default value: true
selectable

Activate selection capability.

Type: Boolean
Default value: true
sortable

Add sorting capability to this column.

Type: Boolean
Default value: true
editable

Activate edition capability. overrided by ng-datasheet editable property.

Type: Boolean
Default value: true
autoOpen

On click on this column on a row, auto switch in edit mode.

Type: Boolean
Default value: false
cellView

Component used for display of the dataset object property (see Ceels section).

Type: CellDynamicInterface
cellEdit

Component used for edition of the dataset object property (see Ceels section).

Type: CellDynamicInterface
componentParam

Extra parameters used to setup a column (depending of the cell view/edit selected).

Type: Object

Outputs

itemEvent

Event fired after finnish editing of a cell of this column.

Type: ItemEvent
