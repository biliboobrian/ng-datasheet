NgDatasheet

Component

The NgDatasheet is the main component to instanciate a grid.

Selector ds-datasheet

Exported as NgDatasheetComponent

Inputs

dataSet

An array of object representing data display in datasheet.

Type: Object[]
columns

Set of columns display in the datasheet

Type: Column[]
parameterButtons

Set of aditionnal link on the global menu on up/right corner. used to apply an action to a set of selected row.

Type: ParameterButton[]
defaultTranslation

Default translation for all words on the datasheet.

Type: DefaultTranslation
withFilters

Global flag to use the filters on columns.

Type: Boolean
Default value: true
withAdd

Global flag to insert a "new line on the datasheet to insert directly at end of dataset a new item.

Type: Boolean
Default value: true
newModelFunction

A given function to return an empty object for "new line" > valid only with withAdd set to true.

Type: Function
dateFormat

Date format for datePicker cells.

Type: String
Default value: MM-DD-YYYY
static

Set if the dataset is a complete set of data or a part only

  • in case of static equal false filters, sort and pagination will not apply to the dataset, only a renderEvent with fire on every change.
  • if static equal false, filters, sorting and pagination will apply directly to dataset.
Type: Boolean
Default value: true
withPagination

Add a pagination to your datasheet > cannot be used with withAdd flag set to true.

  • in case of static equal false filters, sort and pagination will not apply to the dataset, only a renderEvent with fire on every change.
  • if static equal false, filters, sorting and pagination will apply directly to dataset.
Type: Boolean
Default value: false
filters

Initial filters apply to the datasheet, this property is given on renderEvent fire.

Type: Filter[]
pagination

Initial pagination apply to datasheet.

Type: Pagination
sort

Initial sorting apply to datasheet.

Type: Sort
dsKey

If set, dsKey store in local storage informations about columns user preset.

Type: String
trBgColor

Background color apply to row on mouse over.

Type: String
editable

Set if datagrid can be globally editable

  • if editable equal true, in case of column non editable, column will be grey en disable.
  • if editable equal false, all columns will be view only and non of them will be in grey background.
Type: Boolean
Default value: true
globalMenu

Set if up/right global menu with row selection column under it.

Type: Boolean
Default value: false

Outputs

renderEvent

RenderEvent fire every change on sort, filters or pagination modifications.

Type: RenderEvent

Methods

updateDatasheet

updateDatasheet() => void

Reapply all column render to the datasheet, when you change column set for exemple.
